## Important Next Steps

### 1. Configure Contact Form (REQUIRED)
The contact form uses Web3Forms (free service). You need to:

1. Go to https://web3forms.com/
2. Sign up using **contactus@kynowa.com** email
3. Get your free access key from the dashboard
4. Open `src/components/contactform.astro`
5. Replace `YOUR_ACCESS_KEY_HERE` on line 13 with your actual access key

**Without this step, the contact form will not work!**

### 2. Update ABN (When Available)
When you receive your ABN:
- Edit `src/components/footer.astro`
- Replace `ABN: [Pending]` with your actual ABN number

### 3. Install Dependencies & Run Development Server

```bash
cd C:\Claude\Design\root\gitclone\astroship

# Install dependencies (use pnpm, npm, or yarn)
pnpm install
# or
npm install

# Start development server
pnpm dev
# or
npm run dev
```

Visit http://localhost:4321 to see your site locally.

### 4. Optional Customizations

#### Add Team Members
The About page has a team section. To add team members:
1. Add team member markdown files in `src/content/team/`
2. Follow the format of existing team member files
3. Add profile images to the appropriate directory

#### Remove Unused Pages
The template includes pages you might not need:
- `src/pages/pricing.astro` - Delete if not offering pricing tiers
- `src/pages/blog.astro` and `src/pages/blog/[slug].astro` - Delete if not maintaining a blog
- Team content in `src/content/team/` - Remove or replace with actual team

#### Update Images
Replace placeholder images:
- `public/favicon.svg` - Your company favicon
- `public/opengraph.jpg` - Social media preview image
- `src/assets/hero.png` - Main hero image (consider replacing with IT-themed image)

### 5. Deploy to Vercel

When ready to deploy:

```bash
# Build the site
pnpm build

# Preview the build
pnpm preview
```

#### Deployment Options:

**Option A: Deploy via Vercel Dashboard**
1. Push your code to GitHub
2. Go to https://vercel.com
3. Import your GitHub repository
4. Vercel will auto-detect Astro and deploy

**Option B: Deploy via Vercel CLI**
```bash
npm i -g vercel
vercel
```

**Update Domain:**
- Edit `astro.config.mjs` line 9 to update from `https://kynowa.com` to your actual domain when deployed

### 6. SEO & Analytics (Recommended)

#### Add Google Analytics
1. Get your GA4 tracking ID
2. Add it to `src/layouts/Layout.astro` in the `<head>` section

#### Update robots.txt
The file `public/robots.txt` is configured. Update it if you have specific crawling requirements.

#### Create a sitemap
The sitemap is automatically generated by Astro. It will be available at `https://yourdomain.com/sitemap-index.xml` after deployment.
